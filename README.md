Memoroid

Project Description
A meme scrolling app that pulls memes from reddit, and allows you to create your own.

Technologies Used
Kotlin  
Gradle 
JUnit/ Mockito

Features

Scroll memes drawn from Reddit API
All standard list features
Creating memes that will superimpose text over an image
